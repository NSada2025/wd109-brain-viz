# WD109: Layered 3D Brain Map

インタラクティブな3D脳可視化システム - ドーパミン神経系の統合的表現

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

## 機能

- **3D脳モデル**: 透明度調整可能な脳構造表示
- **ドーパミン経路**: VTA/SNcからの主要投射経路
- **電気活動**: 各周波数帯域の波形アニメーション
- **インタラクション**: マウス操作、キーボードショートカット

## 操作方法

### マウス操作
- 左ドラッグ: 回転
- 右ドラッグ: パン
- スクロール: ズーム

### キーボード
- Space: 再生/一時停止
- B: 脳モデル表示切替
- D: ドーパミン経路表示切替
- E: 電気活動表示切替

## 技術スタック

- React + TypeScript
- Three.js (react-three-fiber)
- Zustand (状態管理)
- Leva (UIコントロール)
- Vite (ビルドツール)
